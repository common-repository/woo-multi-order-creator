Generate WooCommerce Orders Product Wise.

== Description ==

Plugin helps to create multiple orders  of multiple product in single shopping.Admin can set delivary days difference and datepicker for delivery date selection.Customer can set delivery date for each product order from checkout page.

= Usage Guide =
- Set order delivery day difference from woocommerce > Multi-order Settings
- Set order delivery day datepicker from woocommerce > Multi-order Settings
- Customer can set order delivery date from checkout page. 

This plugin has been developed to create orders product wise for your WooCommerce website. Any suggestions or feature requests are welcome! 

== Installation ==
- Download the Plugin (in a zip file) on your system
- In the Admin Panel click on Plugin and select ‘Add New’ from the sub menu
- In the next screen Upload the plugin by browsing to the location where you have downloaded the plugin
- Activate the plugin through the 'Plugins' menu in WordPress

Or 

- Upload the woo-multi-order-creator (directory included) to the /wp-content/plugins/ directory
- Activate the plugin through the 'Plugins' menu in WordPress
